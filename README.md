# 🛒 Menu Produk Python

Program sederhana berbasis Python yang menerapkan konsep **if-else**, **input pengguna**, dan **percabangan kondisi** untuk menampilkan daftar produk serta informasi harga berdasarkan pilihan pengguna.

## 📋 Fitur

* Menampilkan menu produk melalui perintah `.menu`
* Menampilkan daftar produk yang tersedia
* Menampilkan harga produk berdasarkan pilihan pengguna
* Validasi perintah menggunakan percabangan `if-else`
* Cocok untuk latihan dasar Python bagi pemula

| No | Produk   | Harga        |
| -- | -------- | ------------ |

| 2  | Mouse    | Rp 100.000   |
| 3  | Keyboard | Rp 250.000   |
| 4  | Monitor  | Rp 1.500.000 |
| 5  | Printer  | Rp 1.200.000 |

## 🚀 Cara Menjalankan

Pastikan Python telah terpasang di komputer.

Cek versi Python:

```bash
python --version
```

Jalankan program:

```bash
python main.py
```
